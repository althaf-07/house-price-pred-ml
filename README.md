![Python](https://img.shields.io/badge/python-3.12-blue)
![Last Commit](https://img.shields.io/github/last-commit/althaf-07/house-price-prediction)

## Project Overview

- **Project Name**: House Price Prediction
- **Objective**: Build a machine learning model to predict house prices.
- **Type of Problem**: Regression
- **Evaluation Metrics**:  RMSE, MAE, and $R^2$ Score

---

## Getting Started

To set up the project and get it running locally, follow the steps in [reports/environment.md](reports/environment.md).

## Project Directory Structure

```bash
house-price-prediction/
├── .git/                        # Git dir
├── .venv/                       # Local Python venv
├── data/                        # All dataset related files
│   ├── interim/                 # Intermediate data after train test split
│   │   ├── test.csv             # Test data split
│   │   └── train.csv            # Train data split
│   ├── processed/               # Processed data after preprocessing
│   │   ├── test.csv             # Processed test data split
│   │   └── train.csv            # Processed train data split for detailed EDA
│   └── raw/                     # Raw and original data
│       └── entire_data.csv      # The original dataset containing both train and test data
├── logs/                        # Logs generated by scripts
├── models/                      # Serialized objects
│   └── pl.joblib                # Pipeline object
├── notebooks/                   # Jupyter notebooks for experimentation
│   └── experimentation.ipynb    # Exploratory and experimental notebook
├── reports/                     # Documentation and result summaries
│   ├── figures/                 # Storage for figures and plots
│   │   └── univariate/          # Figures related to univariate analysis
│   ├── environment.md           # Guide for setting up environment
│   └── experiment_document.md   # Markdown summary of experimentation process
├── src/                         # Source code for the project
│   └── house_price_prediction 
│       ├── plot/                # Plotting scripts
│       │   └── univariate/      # Univariate plots
│       │       └── cat_numd.py  # Categorical and discrete numerical features
│       │       └── numc.py      # Continuous numerical features
│       ├── __init__.py          # Something generated by uv
│       ├── auto_gen_table.py    # Script for automatically generating Markdown tables
│       ├── config.yaml          # Configuration file (paths, hyperparameters, etc.)
│       ├── evaluate.py          # Script for evaluating Model on test data
│       ├── predict.py           # Script for prediction on new data (also includes FastAPI backend)
│       ├── preprocess_data.py   # Script for data cleaning and preprocessing
│       ├── split_data.py        # Script for train-test-split
│       ├── streamlit_app.py     # Streamlit for website frontend
│       ├── train.py             # Script for model training
│       └── utils.py             # Utility functions
├── tmp/                         # Temporary files for testing
├── .envrc                       # direnv file for automatically sourcing venv
├── .gitignore                   # Files and folders to ignore in Git
├── .pre-commit-config.yaml      # Configuration file for precommit hooks
├── .python-version              # Python version used in the project
├── pyproject.toml               # Project and dependency configuration
├── README.md                    # Project overview and instructions
└── uv.lock                      # Dependency lock file for uv
```
